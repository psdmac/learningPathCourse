課程目標

從零開始學習，自己打造自己的創意原型。
為期8堂的前端 HTML CSS ( Javascript) 基礎程式語言課程。
目標為讓設計師可以做出 Profilio、UX researcher 可以讓想法實現、創業家做出微型服務的Prototype。
課程由簡單入門，除了教授基礎的程式語言課程外，也將介紹網路資源，引領學員培養自學的能力。

本課程為初階課程，適合：

  • 有設計規劃經驗，但是沒有程式基礎的設計師、專案經理、UX 研究者
  • 學校只有教怎麼用 Dreamweaver 開發的學生
  • 非資訊相關科系學生，想跨入 Web/App 領域
  • 不知道如何自學網頁開發者

如果你是資深的前後端工程師、網頁設計師、前端開發人員，這些課程對你來說可能會太簡單，
但如果您有意願交流、或對Learning Path的未來目標有興趣，非常歡迎聯繫我們！

預期效益

  • 完成一個靜態的網站，上傳至 GitHub、Dropbox 空間。
  • 運用 jQuery 做出簡單的互動網頁。
  • 了解 HTML/CSS 進而可以修改由 wix、Webflow 等網站產生器產生的頁面。

課程大綱

第一週 - 如何自學前端開發

* 點我觀看課程簡報 *
  • 什麼是前端？
  • 為什麼要學前端？
  • 前端的美好，引發學前端的慾望。
  • 如何自學前端開發。
    1. 線上教學資源
    2. 書籍
    3. 每天需要追蹤的資訊
    4. 哪些可以參加的活動
  • 網站產生器
  • 前端開發工具簡介

需要準備

  • 一個好學的心
  • 筆記型電腦（Mac 系統佳）

第二週 - 製作第一個靜態網頁

  • HTML 入門
    1. 標籤 Tag
    2. 標題 Heading、段落 Paragraph
    3. 清單 List
    4. 連結 Link
    5. 圖片 Image
    6. 屬性 Attribute
    7. 表單 Form
    • 介紹 HTML5-boilerplate
    • 無語意標籤 span, div
  • CSS 入門
    • 如何套用 CSS
    • 選擇器 Selector、內容 Property、值 Value
    • display (inline, block, inline-block)
    • box-model (margin, padding, border)
  • Git 入門
    • 什麼是版本控制
    • 在 GitHub 上建立一個 Respository
    • 將 Respository 下載到本機
    • 將建立的檔案加入 git add
    • 提交加入的檔案 git commit
    • 建立分支 git branch
    • 更新遠端 Respository git push

需要準備

  • 安裝好 Sublime Text 2 編輯器，或自己找一款順手的編輯器。
  • 安裝好 Git
  • 預習 HTML/CSS
    • HTML Dog - Beginner 部分
    • Learn to Code HTML & CSS - Lesson 1 ~ Lesson 3
    • GA DASH - Project 1

第三週 - 排版的技巧

  • HTML
    • HTML5 語意標籤
  • CSS
    • box-model
    • box-sizing (content-box, padding-box, border-box)
    • float
    • 單欄式、雙欄式、三欄式排版
    • position (fixed, relative, absoluate)
    • background
  • Devoloper Tool 使用技巧

需要準備

  • 預習
    • Learn to Code HTML & CSS - Lesson 5 ~ Lesson 6
    • 學習 CSS 版面配置
    • GA DASH - Project 2
    • CodeSchool - Discover DevTool Level 1 ~ Level 2

第四週 - 讓網頁動起來

  • jQuery 入門
    • 選擇器
    • jQuery 修改內容、樣式
    • 製作下拉選單
  • CSS3 animate
    • animate
    • transition
    • transform
  • jQuery 搭配 CSS 做出動畫選單

需要準備

  • 預習

第五週 - CSS 從入門到精通

  • CSS 選擇器
    • *
    • A + B
    • A > B
    • nth-child
  • CSS 偽元素
    • hover
    • before, after
    • a single div
  • CSS3 效果
    • box-shadow, text-shadow

需要準備

  • 預習

第六週 - 網站製作實務

  • icon-font
  • 假文產生器
  • 假圖產生器
  • Google font
  • Justfont
  • 用偽元素做出對話框

  • 客製化 Google 表單做出精美問卷
