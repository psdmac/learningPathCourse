第二週 - 製作第一個靜態網頁

  • HTML 入門
    1. 標籤 Tag
    2. 標題 Heading、段落 Paragraph
    3. 清單 List
    4. 連結 Link
    5. 圖片 Image
    6. 屬性 Attribute
    7. 表單 Form
    • 介紹 HTML5-boilerplate
    • 無語意標籤 span, div
  • CSS 入門
    • 如何套用 CSS
    • 選擇器 Selector、內容 Property、值 Value
    • display (inline, block, inline-block)
    • box-model (margin, padding, border)
  • Git 入門
    • 什麼是版本控制
    • 在 GitHub 上建立一個 Respository
    • 將 Respository 下載到本機
    • 將建立的檔案加入 git add
    • 提交加入的檔案 git commit
    • 建立分支 git branch
    • 更新遠端 Respository git push

需要準備

  • 安裝好 Sublime Text 2 編輯器，或自己找一款順手的編輯器。
  • 安裝好 Git
  • 預習 HTML/CSS
    • HTML Dog - Beginner 部分
    • Learn to Code HTML & CSS - Lesson 1 ~ Lesson 3
    • GA DASH - Project 1
